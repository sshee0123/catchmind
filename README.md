# [ 네트워크 프로그래밍 ] 캐치마인드 게임  

java 로 서버, 클라이언트 구현

## 게임 설명  
상대방의 그림을 보고 정답을 맞추는 게임

## 기능  
채팅하기 기능
4명 동시 Play
그림판 기능(전체 지우기, 지우개, 색 여러 개, 굵기변경 등) 
점수 관리 기능
배경음악 on/off 기능
스크린샷 기능
출제자에게 그림 힌트 제공 기능
